About PiscisOS

PiscisOS is a tiny operation system based x86 architecture. This system is developed with assembly language(every piece of the code). Features of PiscisOS:

  • Run in x86 protect model
  • Multi-task supported (Maximum 250 processes)
  • Ram Disk file system (FAT12)
  • System calls
  • Application supported

PiscisOS is a very simple system, I write it just for learning some knowledge about x86 protect model

Dvelopment Environment

PiscisOS is totally developed with assembly language.

  • Flat Assembler (as assembler)
  • Notepad++ (as code editor)
  • mtools (for burning image files)
